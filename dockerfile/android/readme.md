## FAQ:

### M1 Mac 运行报错问题:

- `CPU` 架构问题: `m1 arm64` vs `x86_64`
- 需要构建 `amd64` 版本的镜像: `docker buildx build --platform linux/amd64 --load`
- dockerfile 内指定 `platform` 版本: `FROM --platform=linux/amd64 ubuntu:20.04`
