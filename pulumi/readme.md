## ​Pulumi 是什么？​​

- Pulumi 是一款​​基础设施即代码（Infrastructure as Code, IaC）工具​​，
- 允许开发者使用通用编程语言（如 TypeScript、Python、Go、C# 等）定义和部署云基础设施资源（如虚拟机、数据库、Kubernetes 集群等）。
- 其核心目标是简化云基础设施的管理，同时利用现代编程语言的特性（如类型检查、代码复用、测试等）提升效率和可靠性。
